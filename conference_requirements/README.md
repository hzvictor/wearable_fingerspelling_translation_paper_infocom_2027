# Conference Requirements Cache

This folder keeps the local submission-rule cache used while expanding the CHASE workshop paper into a longer conference submission.

Current working target:

- Venue assumption: IEEE INFOCOM main-conference style submission.
- Rule source: INFOCOM 2026 public author instructions, because INFOCOM 2027 instructions were not publicly reachable from this environment on 2026-06-15.
- Required follow-up: recheck INFOCOM 2027 once its submission site is public.

Files:

- `infocom_main_submission_checklist.md`: page limit, format, double-blind, ethics, and metadata requirements.
- `originality_and_plagiarism_checklist.md`: duplicate-submission, self-plagiarism, prior-work, and citation-risk checklist for expanding the workshop paper.
- `source_urls.md`: official URLs and access notes.
